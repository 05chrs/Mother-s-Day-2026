# Mother’s Day 2026 — kawaii card

A single static page: open [`index.html`](index.html) locally or host it on **GitHub Pages**.

## GitHub Pages

1. Push this repo to GitHub (default branch `main` with `index.html` at the **root**).
2. In the repo on GitHub: **Settings → Pages**.
3. Under **Build and deployment**, set **Source** to **Deploy from a branch**.
4. Choose branch **`main`** and folder **`/` (root)**, then save.

After a minute or two, the site is available at:

**https://05chrs.github.io/Mother-s-Day-2026/**

(Replace `05chrs` / `Mother-s-Day-2026` if your username or repo name differs.)

## Local preview

Open `index.html` in a browser, or from this folder run:

```bash
python3 -m http.server 8080
```

Then visit `http://localhost:8080`.
